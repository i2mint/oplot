oplot.matrix.hierarchical_cluster_sorted_heatmap(df, only_return_sorted_df=False, seaborn_heatmap_kwargs=None)[source]¶

A function to plot a square df (i.e. same indices and columns) that contains distances/similarities as it’s values, as a heatmap whose indices and columns are sorted according to a hierarchical clustering (based on the distances listed in the df).

Parameters

  • df – The distance (or similarity) square matrix

  • only_return_sorted_df – Default False. Set to True to return the df instead of the heatmap

  • seaborn_heatmap_kwargs – the arguments to use in seaborn.heatmap (default is dict(cbar=False))

Returns

whatever sns.heatmap returns, or the sorted df if only_return_sorted_df=True
